\# Food Delivery Web Application 🍔



A Swiggy-inspired Food Ordering Web Application developed using \*\*Java Servlets, JDBC, MySQL, HTML, CSS, and Apache Tomcat\*\*.



The application allows users to register, log in, browse restaurants, view menus, add items to a cart, make payments, and place orders seamlessly.



\## 🚀 Features



\### User Management



\* User Registration

\* User Login \& Logout

\* Session Management

\* Profile Management



\### Restaurant \& Menu



\* View Restaurants

\* Browse Restaurant Menus

\* Display Food Images and Details



\### Cart Management



\* Add Items to Cart

\* Update Item Quantity

\* Remove Items from Cart

\* Clear Cart



\### Order Management



\* Place Orders

\* View Order History

\* Order Confirmation



\### Payment System



\* UPI Payment

\* Credit Card Payment

\* Debit Card Payment

\* Cash on Delivery (COD)
